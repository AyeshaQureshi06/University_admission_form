# **App Name**: Verdant Admissions

## Core Features:

- Applicant Form: A multi-step form for applicants to submit their personal, academic, and program information.
- Document Upload: Allow applicants to upload required documents such as CNIC, certificates, and photographs to Supabase Storage.
- Application Tracker: Enable applicants to track their application status using their Application ID or CNIC.
- Admin Dashboard: Provide an admin interface for managing applications, staff users, and generating reports.
- Staff Management: Allow administrators to manage staff users and their roles (admin, reviewer, data entry).
- Automated Notifications: Send automated email notifications to applicants upon submission and status changes, and to admins for new applications, leveraging a tool that intelligently alerts the admissions department to exceptional applicant submissions.
- Database Integration: Store and retrieve applicant data, documents, and reviews using Supabase PostgreSQL.

## Style Guidelines:

- Primary color: Deep forest green (#1A472A) to evoke a sense of nature and growth.
- Accent color: Soft mint white (#F0F7F4) to provide a clean and modern contrast.
- Highlight color: Vivid emerald (#2D9A5F) for CTAs and progress indicators.
- Headline font: 'Cormorant Garamond' (serif) for elegant headings.
- Body font: 'Plus Jakarta Sans' (sans-serif) for clean and modern body text.
- Use green-themed icons related to application steps and data types.
- Employ clean white cards with soft green left-border accents for a structured layout.
- Use subtle transitions and animated checkmarks/confetti for positive feedback.
